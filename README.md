🃏 Jogo da Memória - Marvel vs DC

Projeto desenvolvido com HTML, CSS e JavaScript puro, utilizando manipulação de DOM, animações 3D e lógica de jogo para criar um sistema interativo de cartas da Marvel e DC.

🚀 Funcionalidades

🎮 Sistema completo de jogo da memória
🔄 Embaralhamento automático das cartas
🦸 Troca dinâmica de temas:
Marvel
DC
🎴 Animação Flip 3D nas cartas
📊 Contador de tentativas
🔒 Bloqueio de múltiplos cliques
📱 Estrutura inicial responsiva
✨ Efeitos visuais com CSS

🎯 Como Funciona

O jogador deve encontrar os pares corretos das cartas.

Quando duas cartas iguais são selecionadas:

✅ Elas permanecem viradas

Quando são diferentes:

❌ Elas voltam para a posição inicial após alguns segundos

O sistema também contabiliza o número de tentativas realizadas durante a partida.
